Local Decisioning API Reference

This is a proposal for the Adobe Target Local Decisioning API.

Decisions

Get Offers

adobe.target.getOffers({
  request: {
    execute: {
      mboxes: [{
        index: 0,
        name: "ui"
      },{
        index: 1,
        name: "pricing"
      }]
    }
  },
  allowRemote: false
})
.then(response => {
  ...
}

getOffers(options)

Options Parameters

Parameter Default Description
request required Existing request object for a getOffers() call. See existing documentation.
allowRemote true If false, only perform local decisioning results. Useful when latency is critical and local decisioning is required. Otherwise, if true and every offer in the request cannot be calculated locally, a request will be made to Adobe Target servers.

Feature Flags

This is a high-level call that will perform a getOffers() call to retrieve JSON content for one mbox and convert it into an object that encapsulates feature flags as either a simple JSON/javascript object or can be called to retrieve type-specific values. This is useful for the use-case when you are using a single mbox for all feature flags.

adobe.target.getFeatureFlags("feature-flags", { allowRemote: false }).then(flags => {

  // You can just convert to simple javascript object and use that

  const featureFlags = flags.asObject();
  if (featureFlags["hide_price"]) {
    const priceBox = document.getElementById("price-box");
    if (priceBox) {
      priceBox.style.display = 'none';
    }
  }

  // OR, use type-specific methods to retrieve values

  const hidePayment = flags.getFeatureBoolean("hide_payment");
  if (hidePayment) {
    const paymentBox = document.getElementById("payment-box");
    if (paymentBox) {
      paymentBox.style.display = 'none';
    }
  }

getFeatureFlags(mbox, options)

Parameters

Parameter Default Description
mbox required Name of mbox that contains feature flags encoded as JSON content.
options {} Options are passed into getOffers(), see SDK-specific documentation.

FeatureFlags object returned

  • asObject()

Alternatively, you can use the object to return type-specific values given a key:

  • getFeatureBoolean(key)

  • getFeatureString(key)

  • getFeatureInteger(key)

  • getFeatureFloat(key)

      }]
    }
  }
})

  ...
}

This lower-level call returns offers that will only be calculated locally. This call is the equivalent of calling getOffers() with the allowRemote=false flag.
